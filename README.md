<h1>React Assignment<h1>
<p>In this assignment React is used make simple todo list application which includes basic functions as adding a task, deleting a task, editing a task. It uses react components with props and event handling. </p>

<h1>Objective of this project are:</h1>
<ol>
    <li>To develop a todo list application</li>
    <li>To use react for bulding the application</li>
    <li>To define necessary react components</li>
    <li>To use hooks to make the static webpage dynamic</li>
    <li>To have a feature to add new task to the list</li>
    <li>To have a feature to delete a task</li>
    <li>To have a feature to edit an existing task on the list</li>
    <li>To have a feature that indicates task is completed</li>
</ol>

<h1>How to deploy and run this application</h1>
<ul>
    <li>How to clone the application: git clone https://github.com/rijulsrivastava/react-assignment.git </li>
    <li>Then open terminal</li>
    <li>locate src folder in the application</li>
    <li>run command: npm run dev</li>
</ul>
